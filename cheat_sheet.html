Busquedas importantes

Usar Emmet

Emmet esta preconfigurado en VS code desde la isntalacion y ayuda a escribir html y sus variaciones como haml, jade, slim, jsx, xml, xsl, css, scss, sass, less y stylus

.my-div>p.my-text

======>
<div class="my-div"> <p class="my-text"></p> </div>

Terminal integrada

Navegar por definiciones

Podemos movernos entre las definiciones y referencias de "funciones" solo dando click derecho y referenciando las opciones

peek definition

Intelisence

Normalmente funciona de forma automatica, pero tambien se puede llamar por comando

Debbuger

nos ayuda a chequear valores en nuestros scripts y en general a hacer testing (por defecto funciona con nodeJS pero se pueden configurar mas lenguajes/frameworks)

Integracion con git
